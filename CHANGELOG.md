# Changelog

All notable changes to this project will be documented in this file.

## [v1.0.0] - 2026-02-14

### Added
- **High-Performance Rendering**: Implemented PyQt6/QWebEngine core for fast Markdown display.
- **GitHub-Style UI**: Clean, modern interface mirroring GitHub's rendering.
- **Dynamic Theme Support**: Automatic Dark/Light mode switching based on system preference.
- **Window Management**: Application auto-starts in "Maximized" mode.
- **Build System**:
    - `build.py`: Automated PyInstaller build script with artifact cleanup.
    - `make_setup.py`: Interactive Inno Setup integration for installer generation.
- **Windows Integration**:
    - Context menu support.
    - Registry association for `.md` files.
    - Custom Control Panel uninstall icon.
- **Localization**: Installer support for English and Turkish.
